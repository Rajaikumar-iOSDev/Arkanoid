# Arkanoid

Arkanoid game built with Raylib based on the Ankanoid example on Raylib repo(https://github.com/raysan5/raylib-games/blob/master/classics/src/arkanoid.c).



## How to run
Clone the repo
cd Arkanoid!
zig build run
